VSR_to_datetime¶

DatesTimes.VSR_to_datetime(VSR_time_tuple)¶

Converts a VSR time tuple to a datetime object. Example:: >>> DateTimes.VSR_to_datetime((2010,15,16212)) >>> datetimes.datetime(2010, 1, 15, 4, 30, 12) To create a VSR time tuple from a string use VSR_to_timetuple
